# HiliteHttp
HiliteSDK Http module - facilitates communication with the HILITE api

## Integration

#### CocoaPods (iOS 10+)

Install HiliteHttp via CocoaPods by adding it to your `Podfile`:

```ruby
platform :ios, '10.0'
use_frameworks!

target 'MyApp' do
    pod 'HiliteHttp', '~>0.1.4-alpha'
end
```

## Usage

import HiliteCore
import HiliteHttp

}
